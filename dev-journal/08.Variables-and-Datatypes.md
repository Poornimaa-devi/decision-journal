# Day 8 – Variables and Data Types

## Today's Goal

Learn how variables, data types, and objects are used to represent information in JavaScript and understand how they relate to my Decision Journal project.

---

## What I Learned

Today I learned the difference between primitive and non-primitive data types. I also understood that objects help group related information together, making the code more organized.

I created sample User and Goal objects and displayed them in the browser console.

---

## Most Confusing Topic

Understanding variable scope was slightly confusing at first. I learned that scope determines where a variable can be accessed, which helps prevent unintended changes to data.

---

## Where I'll Use Today's Concepts in My Project

I will use variables to store values such as goal progress and user input.

Objects will represent the main entities of my application, including users, goals, reflections, and milestones.

As the project grows, these objects will be stored in React state and later saved in MongoDB.

---

## Reflection

Today's lesson helped me think like a software engineer by modeling real-world information before writing code. I now understand why objects are essential for building scalable applications.
