# Day 4 – HTML Landing Page

## Today's Goal

Learn the fundamentals of HTML and create the initial structure of the Decision Journal landing page.

---

## What I Built

Today I created the first webpage of my project using HTML.

The landing page includes:

- Header
- Navigation
- Hero section
- About section
- Features section
- Footer

I focused only on organizing the content using semantic HTML elements without worrying about styling.

---

## What I Learned

Today I learned that HTML provides the structure of a webpage. It tells the browser what content should exist but not how it should look.

I also understood the importance of semantic HTML and how it improves readability, accessibility, and SEO.

I learned that React is built on top of HTML concepts, making a strong understanding of HTML essential before learning React.

---

## One Mistake I Made

Initially, I wanted to use generic `<div>` elements for everything. After learning about semantic HTML, I replaced them with meaningful tags such as `<header>`, `<main>`, `<section>`, and `<footer>` to make the page structure more organized.

---

## How HTML Will Help Me in React

React components ultimately generate HTML elements.

Understanding HTML will help me build better React components, organize page layouts effectively, and create more accessible user interfaces.

The HTML structure I created today will later become reusable React components in the Decision Journal application.
