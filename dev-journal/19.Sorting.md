# Day 19 – Development Journal

## Today's Goal

To implement sorting so users can view goals in different orders without changing the original data.

---

## What I Built

- Added sorting functionality.
- Added a Sort dropdown.
- Implemented A–Z and Z–A sorting.
- Combined Search, Filter, and Sort into one workflow.

---

## Biggest Learning

I learned that sorting should only affect the displayed data and not the original goals array.

---

## Hardest Concept

Understanding why a copy of the array should be sorted instead of modifying the original array.

---

## How Sorting Improved My Project

Sorting makes it easier for users to organize their goals and find information quickly. It also improves the overall user experience.
