# Day 12 – Forms and Validation

## Today's Goal

Learn how forms collect user input and use JavaScript to create new goals without refreshing the page.

---

## What I Built

Today I created my first goal creation form with fields for the goal title, deadline, and priority.

When the user submits the form, JavaScript reads the input values, creates a goal object, stores it in the goals array, and updates the webpage dynamically.

---

## Biggest Challenge

Understanding why `preventDefault()` is needed was initially confusing. After practicing, I realized it prevents the browser from refreshing the page so JavaScript can handle the form submission.

---

## What Surprised Me

I was surprised that I could create new HTML content dynamically using only JavaScript without manually writing additional HTML.

---

## Where I'll Use Forms Later

I will use forms for:

- User registration
- User login
- Creating goals
- Editing goals
- Writing reflections
- Updating progress

Forms will become one of the most important parts of my Decision Journal application.

---

## Reflection

Today's lesson helped me understand how users interact with web applications. Creating goals through a form made my project feel much more interactive and closer to a real-world application.
