# Day 5 – CSS Styling

## Today's Goal

Learn the fundamentals of CSS and transform the basic HTML structure of the Decision Journal landing page into a visually appealing interface.

---

## What I Styled Today

Today I created a separate CSS file and linked it to my HTML page.

I styled the following sections:

- Header
- Navigation bar
- Hero section
- Call-to-action button
- Feature cards
- Footer

I also selected a simple color palette and maintained consistent spacing throughout the page.

---

## What CSS Concept Confused Me

Initially, I found the difference between margin and padding confusing because both create space.

After experimenting with different values in the browser, I understood that margin creates space outside an element, while padding creates space inside it.

---

## How I Solved It

I practiced using different margin and padding values on buttons and containers. Seeing the visual changes helped me understand the difference much better than simply reading definitions.

## Thinking Exercise

If my landing page looks perfect on my laptop but overlaps on a phone, it means the layout was designed only for one screen size.

Responsive design solves this problem by allowing the webpage to adjust its layout based on the screen size, ensuring a good user experience across all devices.
