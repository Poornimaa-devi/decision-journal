# Day 17

## Today's Goal

Learn how to save application data using Local Storage so that goals remain available after refreshing the page.

---

## What I Built

- Saved goals to Local Storage.
- Loaded saved goals when the application started.
- Used JSON to store arrays of objects.
- Made the application remember user goals after refresh.

---

## Biggest Learning

I learned that JavaScript variables exist only while the page is running. To keep data permanently, it must be stored in Local Storage or a database.

---

## Most Confusing Topic

Understanding why objects must be converted into JSON before saving them in Local Storage.

---

## How Local Storage Improved My Project

Before today, refreshing the page erased all my goals.

Now the browser remembers my goals automatically, making the application feel much more realistic.

---
