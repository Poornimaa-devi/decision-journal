## Most Interesting Concept

The most interesting concept was event handling.

I learned that JavaScript waits for user actions such as clicking a button and then executes a specific function.

This showed me how HTML and JavaScript communicate with each other.

---

Today I learned that JavaScript adds behavior to websites.
