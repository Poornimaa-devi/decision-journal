# Day 11 – DOM Manipulation

## Today's Goal

Understand how JavaScript interacts with HTML using the Document Object Model (DOM) and display data dynamically on the webpage.

---

## What I Built

Today I added a **My Goals** section to the landing page.

Instead of writing each goal directly in HTML, I stored the goals in a JavaScript array and used DOM manipulation to display them automatically.

---

## Most Interesting Concept

The most interesting concept was creating HTML elements using JavaScript. I learned that webpages can generate content dynamically instead of relying only on static HTML.

---

## Biggest Challenge

Understanding how JavaScript creates elements and adds them to the webpage using `createElement()` and `appendChild()` was initially challenging, but practicing with my goals array made the process clearer.

---

## What Excited Me Today

I was excited to see my webpage update itself using data from JavaScript. It made the project feel more like a real application rather than a static webpage.

---

## Reflection

Today's lesson helped me understand how modern web applications display dynamic data. This concept will be very useful when I start learning React, where components automatically render data from JavaScript.
