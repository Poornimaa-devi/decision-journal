# Day 10 – Arrays and Objects

## Today's Goal

Learn how arrays store collections of data and understand how arrays and objects work together to represent real-world information.

---

## What I Built

Today I created an array named `goals` containing multiple goal objects.

I practiced:

- Creating an array of objects.
- Finding the total number of goals using `.length`.
- Looping through the array using `forEach()`.
- Printing each goal title to the browser console.

---

## Most Useful Concept

The most useful concept I learned today was using an array of objects.

I realized that every goal can be represented as an object, and all the goals can be stored inside one array. This structure closely matches how data is managed in real-world applications.

---

## How Arrays Will Help My Project

Arrays will allow my application to manage multiple goals, reflections, and milestones.

Instead of creating separate variables for each goal, I can store them all in one array and easily add, remove, update, or display them.

This approach will make the Decision Journal scalable and easier to maintain as more features are added.

---

## Reflection

Today's lesson helped me understand how data is organized in modern web applications. I now see how arrays and objects work together to model real-world information in a clean and structured way.
