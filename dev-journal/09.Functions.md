# Day 9 – Functions

## Today's Goal

Understand how functions help organize JavaScript code and make it reusable.

---

## What I Built

Today I improved my JavaScript code by organizing it into reusable functions.

I created separate functions for:

- Showing the welcome message.
- Displaying the login message.
- Displaying the signup message.

Each function has a single responsibility, making the code easier to understand.

---

## What Was Difficult?

At first, I found it confusing why we pass a function to `addEventListener()` instead of calling it immediately.

After practicing, I understood that JavaScript waits until the event occurs before executing the function.

---

## How Functions Improved My Code

Functions made my code cleaner and prevented duplication.

Instead of writing the same logic every time a button is clicked, I created reusable functions that can be called whenever needed.

This approach makes the application easier to maintain and expand.

---

## One Thing I Still Need to Practice

I want to practice writing functions with parameters and return values.

I also want to become more comfortable using arrow functions since they are widely used in modern JavaScript and React.

---

## Reflection

Today's lesson helped me understand that functions are one of the most important building blocks of programming. They allow me to break large problems into smaller, reusable pieces, making my code more organized and scalable.
