# Day 16

## Today's Goal

Implement the Delete operation for goals and complete the CRUD functionality of the Decision Journal application.

---

## What I Built

- Added a Delete button for every goal.
- Displayed a confirmation dialog before deletion.
- Removed the selected goal from the goals array.
- Re-rendered the updated goal list after deletion.

---

## Biggest Challenge

Understanding how to remove the correct goal without affecting the other goals and ensuring the UI updated correctly after deletion.

---

## How Delete Works in My Application

1. User clicks the Delete button.
2. A confirmation dialog appears.
3. If confirmed, the selected goal is removed from the goals array.
4. renderGoals() refreshes the webpage.
5. The deleted goal no longer appears.

---

## What Surprised Me Today

I realized that I never needed to remove HTML elements manually. Updating the data and calling renderGoals() automatically updated the user interface.
