# Day 2 – Understanding How the Internet Works

## Today's Goal

Learn the fundamentals of how browsers communicate with servers and understand the basic architecture of a web application.

---

## What I Learned

Today I learned that every web application follows a client-server model. The browser acts as the client and communicates with the backend server using HTTP requests. The server processes those requests, interacts with the database if needed, and returns a response to the browser.

I also understood that the frontend is responsible for the user interface, while the backend manages business logic and data storage.
