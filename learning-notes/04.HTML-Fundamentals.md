# HTML Fundamentals

## What is HTML?

HTML (HyperText Markup Language) is the standard language used to create the structure of web pages. It defines the content that appears on a webpage, such as headings, paragraphs, images, buttons, forms, and links.

HTML does not control the appearance or behavior of a webpage. Instead, it provides the foundation upon which CSS and JavaScript work.

---

## Why Does HTML Exist?

Browsers need a common language to understand what should be displayed to users. HTML provides this language.

Whenever a user visits a website, the server sends an HTML document to the browser. The browser reads the document and displays its contents as a webpage.

Without HTML, browsers would not know how to organize or display information.

---

## Browser Rendering Process (High Level)

The process of displaying a webpage is simple at a high level:

1. The user enters a website address in the browser.
2. The browser sends a request to the server.
3. The server returns an HTML document.
4. The browser reads the HTML from top to bottom.
5. The browser creates the webpage based on the HTML structure.
6. CSS styles the page.
7. JavaScript adds interactivity.

---

## Semantic HTML

Semantic HTML uses tags that clearly describe the purpose of their content.

Examples include:

- `<header>`
- `<nav>`
- `<main>`
- `<section>`
- `<article>`
- `<aside>`
- `<footer>`

These tags make the code easier to understand and improve accessibility and search engine optimization (SEO).

---

## Why Shouldn't We Use `<div>` Everywhere?

A `<div>` is a generic container with no meaning.

If every part of a webpage is built using only `<div>` elements, it becomes difficult for developers, screen readers, and search engines to understand the page structure.

Semantic elements clearly communicate the purpose of each section, making the application more maintainable and accessible.

---

## How React Uses HTML

React does not replace HTML. Instead, React uses JSX, which closely resembles HTML syntax.

When a React component is rendered, React converts JSX into HTML elements that the browser can display.

Understanding HTML makes it much easier to learn React because every React component ultimately produces HTML.
