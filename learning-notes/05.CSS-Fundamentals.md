# CSS Fundamentals

## What is CSS?

CSS (Cascading Style Sheets) is a stylesheet language used to control the appearance of HTML elements. While HTML provides the structure of a webpage, CSS is responsible for its presentation, including colors, fonts, spacing, layouts, and responsiveness.

Without CSS, websites would only display plain text and basic HTML elements without any visual design.

---

## Why Does CSS Exist?

HTML tells the browser **what** should be displayed, while CSS tells the browser **how** it should be displayed.

Separating structure from presentation makes web pages easier to maintain and update. It also allows the same HTML structure to have different designs without changing the content.

---

## CSS Selectors

Selectors determine which HTML elements a CSS rule applies to.

### Element Selector

Targets all elements of a specific type.

Example:

```css
h1 {
    color: navy;
}
```

---

### Class Selector

Targets multiple elements that share the same class.

Example:

```css
.hero {
    text-align: center;
}
```

Classes are reusable and are the preferred choice for styling groups of elements.

---

### ID Selector

Targets one unique element.

Example:

```css
#logo {
    font-size: 32px;
}
```

IDs should only be used when an element is unique on the page.

---

## The Box Model

Every HTML element is treated as a rectangular box made up of four parts:

- Content
- Padding
- Border
- Margin

Understanding the box model is essential because it controls the spacing and size of every element on a webpage.

---

## Margin vs Padding

### Margin

Margin creates space **outside** an element, separating it from neighboring elements.

Example:
The margin around a button creates distance between the button and surrounding content.

---

### Padding

Padding creates space **inside** an element, between its content and border.

Example:
Increasing a button's padding makes it larger without changing the surrounding layout.

---

## Flexbox

Flexbox is a CSS layout system that makes it easier to arrange elements horizontally or vertically.

Common Flexbox properties include:

- `display: flex`
- `justify-content`
- `align-items`
- `gap`

In my Decision Journal landing page, Flexbox is useful for arranging navigation links, buttons, and feature cards in a clean and organized layout.

---

## Why Responsive Design Is Important

Users access websites from devices with different screen sizes, such as phones, tablets, laptops, and desktop computers.

Responsive design ensures that a webpage adjusts its layout automatically so that it remains readable and usable on all devices.

Without responsive design, elements may overlap or become difficult to use on smaller screens.
