# DOM Manipulation

## What is the DOM?

The DOM (Document Object Model) is a tree-like representation of an HTML document created by the browser. It allows JavaScript to access and modify the webpage after it has loaded.

---

## Why Does the DOM Exist?

The DOM allows web pages to become interactive. Without it, JavaScript would not be able to change text, update styles, respond to user actions, or create new elements dynamically.

---

## DOM Tree

When a browser reads an HTML file, it converts every HTML element into a tree structure.

Example:

Document
│
└── html
    ├── head
    └── body
        ├── header
        ├── main
        └── footer

JavaScript interacts with this tree to update the webpage.

---

## Selecting Elements

JavaScript must first find an element before it can change it.

Common methods include:

- `getElementById()`
- `querySelector()`

These methods help locate HTML elements for manipulation.

---

## Creating Elements

JavaScript can create new HTML elements while the page is running.

This is useful for displaying dynamic content such as goals, notifications, or user data without writing them manually in HTML.

---

## Event Listeners

Event listeners allow JavaScript to respond to user actions.

Examples:

- Clicking a button
- Typing in an input field
- Submitting a form

Using `addEventListener()` keeps JavaScript separate from HTML and is considered best practice.
