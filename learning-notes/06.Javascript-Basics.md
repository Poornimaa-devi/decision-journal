# JavaScript Basics

## What is JavaScript?

JavaScript is a programming language that adds behavior and interactivity to web pages. While HTML creates the structure and CSS controls the appearance, JavaScript makes a webpage respond to user actions.

---

## Why Was JavaScript Created?

Early websites were static and could only display information. Users could not interact with web pages without reloading them.

JavaScript was introduced to make websites dynamic by allowing developers to validate forms, respond to user actions, update content without reloading the page, and create interactive user experiences.

---

## Variables

Variables are used to store information that a program needs while it is running.

JavaScript provides three ways to declare variables:

### let

Used when the value may change later.

Example:

```javascript
let progress = 50;
```

---

### const

Used when the value should never change.

Example:

```javascript
const appName = "Decision Journal";
```

---

### var

An older way of declaring variables.

Although it still works, modern JavaScript usually prefers `let` and `const` because they are safer and easier to understand.

---

## Data Types

JavaScript stores different kinds of information using data types.

Examples in my Decision Journal project:

- String → Goal title
- Number → Progress percentage
- Boolean → Goal completed or not
- Array → List of goals
- Object → User information
- Null → No reflection added yet
- Undefined → Value not assigned

---

## Functions

A function is a reusable block of code that performs a specific task.

Instead of writing the same code multiple times, we write it once inside a function and call it whenever needed.

In today's project, I created separate functions for the Get Started, Login, and Signup buttons.

---

## Events

Events are actions performed by the user or browser.

Examples include:

- Clicking a button
- Typing in a text box
- Submitting a form

JavaScript listens for these events and executes the appropriate function.

---

## Why JavaScript Is Needed in My Decision Journal

JavaScript allows users to interact with my application.

Examples include:

- Clicking buttons
- Filling forms
- Creating goals
- Updating progress
- Viewing analytics
- Displaying notifications

Without JavaScript, my website would only display information without responding to user actions.
