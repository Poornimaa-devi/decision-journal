# Forms and Validation

## What is a Form?

A form is a group of input fields used to collect information from users. Forms allow users to enter data that can be processed by a web application.

---

## Why Do Forms Exist?

Forms provide a structured way for users to submit information such as login details, feedback, registrations, or new goals. They make web applications interactive by allowing users to contribute data.

---

## What is Form Submission?

Form submission happens when a user clicks the submit button. The browser collects the input values and normally sends them to a server, which causes the page to reload.

---

## Why Use preventDefault()?

The `preventDefault()` method stops the browser's default form submission behavior, preventing the page from refreshing.

This allows JavaScript to process the form data, validate it, update the webpage, and communicate with a backend without interrupting the user's experience.

---

## How JavaScript Reads User Input

JavaScript reads values entered by the user using methods like `getElementById()` and the `.value` property.

These values can then be used to create objects, update arrays, and display new information on the webpage.

---
