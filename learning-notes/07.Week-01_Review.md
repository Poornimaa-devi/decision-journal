# Week 1 Review

## What is HTML?

HTML (HyperText Markup Language) is the foundation of every webpage. It defines the structure and content that the browser displays, such as headings, paragraphs, buttons, images, and forms.

---

## Why Do We Need CSS?

CSS (Cascading Style Sheets) controls the appearance of a webpage. It allows us to style HTML elements by adding colors, fonts, spacing, layouts, and responsive designs. Without CSS, webpages would be plain and difficult to use.

---

## Why Do We Need JavaScript?

JavaScript adds behavior and interactivity to a webpage. It allows users to interact with the application by handling events, validating input, updating content dynamically, and responding to user actions.

---

## How Do HTML, CSS, and JavaScript Work Together?

HTML creates the structure of the webpage.

CSS styles the structure and makes it visually appealing.

JavaScript makes the webpage interactive by responding to user actions.

Together, they create a complete and engaging user experience.

---

## What Was the Most Difficult Concept This Week?

Understanding how JavaScript communicates with HTML through events and functions was initially challenging. Learning how event listeners connect user actions to JavaScript functions helped me understand this concept better.

---

## What Concept Became Clearer After Practice?

The relationship between HTML, CSS, and JavaScript became much clearer after building my own landing page. I understood that each technology has a specific responsibility and they work together to create a functional website.
