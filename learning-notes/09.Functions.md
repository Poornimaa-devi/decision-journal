# Functions in JavaScript

## What is a Function?

A function is a reusable block of code that performs a specific task. Instead of writing the same code multiple times, we write it once inside a function and call it whenever needed.

For example, in my Decision Journal project, I created separate functions for the Get Started, Login, and Sign Up buttons.

---

## Why Do We Need Functions?

Functions help us write clean, organized, and reusable code.

Benefits of using functions:

- Avoid repeating the same code.
- Make programs easier to read.
- Simplify debugging and maintenance.
- Allow different parts of the application to reuse the same logic.

As my Decision Journal grows, functions will help organize features like creating goals, updating progress, and displaying reflections.

---

## Parameters vs Arguments

### Parameters

Parameters are variables defined in a function that receive values when the function is called.

Example:

```javascript
function greetUser(name) {
    console.log("Welcome " + name);
}
```

Here, `name` is the parameter.

---

### Arguments

Arguments are the actual values passed to a function.

Example:

```javascript
greetUser("Poornimaa");
```

Here, `"Poornimaa"` is the argument.

---

## Return Statement

A return statement sends a value back from a function.

Example:

```javascript
function add(a, b) {
    return a + b;
}

let total = add(10, 20);
```

The function returns `30`, which is stored in `total`.

Not every function needs a return value. Some functions simply perform an action, such as displaying a message.

---

## Function Scope

Variables declared inside a function are only available within that function.

This prevents accidental changes to data outside the function and keeps the code organized.

Example:

```javascript
function example() {
    let message = "Hello";
}

console.log(message); // Error
```

---

## Arrow Functions

Arrow functions provide a shorter way to write functions.

Example:

```javascript
const welcomeUser = () => {
    alert("Welcome to Decision Journal!");
};
```

Arrow functions are commonly used in modern JavaScript and React.
